## How to Add a New Product
1. Pick the domain code (wd, hr, ops)
2. Create a 2-4 letter product abbreviation
3. Generate a 1-2 character hex ID
4. Create repo: `kz-[domain]-[product]-[hex]`
5. Add entry to this legend
6. Follow the SOP for setup and deployment
